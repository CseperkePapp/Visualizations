# Example Project Management Surface

This folder is an example of a minimal runnable `docs/project-management/` surface that works with the optional workflow automation kit.

It is intentionally small and meant as a reference, not a required project shape.

## Included

- `tasks/README.md` — task index and next-task counters
- `tasks/TASK-001-example-active-task.md` — an active task
- `tasks/DONE-TASK-000-example-completed-task.md` — a completed task with lifecycle prefix
- `workflows/README.md` — folder-level workflow guidance
- `decisions/README.md` — folder-level decision guidance
- `decisions/DECISION-LOG.md` — minimal decision log example

## How To Use

Copy the folder contents into your project under `docs/project-management/` and adjust naming, links, and conventions to your repo.
